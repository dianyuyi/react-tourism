## 網站相關

### 簡易功能介紹

・搜尋景點：依照景點中的描述，過濾並回傳含有關鍵字的資料。點擊景點會顯示較詳細的說明視窗。

・全台景點：除了全台的總類外，依序分成東、西、南、北、離島，總共六類。分類方式參考台灣觀光局的做法。

・網站相關：資料來源及功能簡易說明。

相關文件及連結請參照官方文件。

### 資料來源

・串接 API：MOTC Transport API V2 / Tourism

・首頁影片：Pixels

提供全台觀光景點的位置及簡易預覽。API 的使用方式及相關連結請參照官方範例說明。
